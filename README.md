# Conexión a la API de Gemini

Este proyecto muestra cómo conectarse a la API de Gemini utilizando Python y la librería `google-genai`.

## Requisitos

- Python 3.10 o superior
- Librerías definidas en `requirements.txt`

## Creación del entorno virtual

```bash
python -m venv env
.\env\Scripts\activate


```bash
pip install - r .\requirements.txt


##Ejecutar 

python .\app_geminy.py
